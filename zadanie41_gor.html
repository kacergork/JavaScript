Zadanie 41 – Teoria pętli

Pętla for

Konstrukcja:

for (let i = 0; i < 5; i++) {
  // instrukcje
}

Opis: Pętla „for” wykonuje blok kodu określoną liczbę razy, znaną z góry.

Pętla while (kontrolowane wejście)

Konstrukcja:

while (warunek) {
  // instrukcje
}

Opis: Pętla „while” wykonuje blok kodu dopóki warunek jest spełniony. Sprawdzenie warunku następuje przed wejściem do pętli.

Schemat blokowy:

Pętla do...while (kontrolowane wyjście)

Konstrukcja:

do {
  // instrukcje
} while (warunek);

Opis: Pętla „do while” zawsze wykona blok kodu przynajmniej raz, ponieważ warunek sprawdzany jest na końcu.
